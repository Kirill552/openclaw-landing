02 / Конфиг

Один конфиг — полный контроль

Модель, каналы, скиллы, плагины, память — всё в одном JSON-файле. Никаких панелей управления, всё прозрачно.

  • Любая модель: Claude, GPT, Gemini, MiniMax, Ollama
  • OpenRouter — не заблокирован в РФ
  • Telegram, VK, MAX, WhatsApp, Discord, Slack
  • Скиллы подгружаются по описанию (description match)
  • Векторная память через LanceDB

06 / FAQ

Частые вопросы

Что такое OpenClaw?
Open-source платформа для создания ИИ-агентов (190K+ звёзд GitHub). Работает на вашем сервере, подключается к Telegram, VK, MAX и другим мессенджерам. Лицензия MIT — бесплатна.
Как установить OpenClaw в России?
OpenClaw работает в РФ через OpenRouter — он не заблокирован. Ставится на VPS (Ubuntu), локальный ПК или Docker. Нужен Node.js 22+, стабильный интернет и API-ключ OpenRouter.
Какие модели поддерживает?
Любые через OpenRouter: Claude, GPT, Gemini, MiniMax, Llama, Kimi. Можно подключить локальные модели через Ollama. Для векторной памяти — text-embedding-3-small через OpenRouter.
Сколько стоит содержать OpenClaw?
Программа бесплатна. Затраты — только API: $1-15/мес через OpenRouter. VPS — от 300 руб/мес. На локальном ПК — только API.
Чем ваша настройка отличается от фриланса за 2000 руб?
На фрилансе продают установку — npm install + базовый конфиг. Мы делаем кастомную настройку: скиллы под задачу, плагины TypeScript, интеграция с VK/CRM/ЮKassa, векторная память LanceDB, hardening по чеклисту безопасности, и долгосрочная поддержка.
